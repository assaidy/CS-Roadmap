## Syntax
```cpp
cout << (condition ? return if true : return if false);
// or 
(condition) ? exec if true : exec if false
```

## Example
```cpp
int main(){
	int num = 10;
	std::cout << (num == 10 ? "num = 10" : "num != 10") << '\n';
	
	return 0;
}
```

```cpp
int main(){
	int num = 10;
	(num == 10) ? std::cout << "num = 10\n" : std::cout << "num != 10\n");
	
	return 0;
}
```
**output**
```txt
num = 10
```
